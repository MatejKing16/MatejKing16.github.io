Reverse Engineering - Aircraft Component

2024 · Academic Project

Overview

Completed a comprehensive reverse engineering project on aircraft exhaust pipe components. The process involved precision measurement using Coordinate Measurement Machine (CMM), complete CAD reconstruction in SolidWorks, and technical documentation per aerospace standards. Demonstrated proficiency in quality assurance measurement techniques, complex surface modeling, and the preparation of aerospace-grade technical drawings following ISO 4706-m standards.

Engineering Process

Measurement & Data Acquisition: Used Coordinate Measurement Machine to capture 10,000+ precision points on the aircraft exhaust pipe. Multi-axis scanning strategy ensured complete surface coverage. Generated point cloud data with ±0.05mm accuracy. Quality verification performed on all measurements using CMM software validation tools.

CAD Reconstruction: Imported point cloud into SolidWorks for surface modeling. Complex curved surfaces reconstructed using NURBS surface fitting techniques. 3D geometry verified against original measurements for accuracy validation. Assembly modeling included all mounting points and interface surfaces.

Aerospace Documentation: Generated technical drawings compliant with ISO 4706-m aerospace standards. Comprehensive documentation includes orthogonal projections, cross-sections, and detailed feature specifications. All dimensions, tolerances, and material callouts per aerospace industry requirements. Final deliverables suitable for manufacturing and quality control purposes.
