AI-Powered CNC Operator

2024 · Side Project

Overview

Developed a web-based AI control system for DIY CNC machines with remote access, computer vision, and intelligent automation. The system uses Google Gemini API for intelligent job planning, OpenCV for real-time visual feedback, and WebSocket for low-latency communication. Fully automated deployment on Linux with PM2 process management and Tailscale VPN for secure remote access.

Technical Architecture

Backend System: Flask web framework with WebSocket support for real-time bidirectional communication. CNCjs integration for machine control. Google Gemini API for intelligent job planning and optimization suggestions. All requests logged and monitored for reliability.

Computer Vision Pipeline: OpenCV-based real-time image processing for workpiece detection and position verification. Automated alignment using fiducial markers. Machine learning models for defect detection. Visual feedback loop ensures accurate tool positioning before each operation.

Automation & Deployment: PM2 process manager handles automatic service restart and monitoring. Tailscale VPN enables secure remote access from anywhere. Health monitoring with automatic error recovery. Fully containerized for easy deployment and scaling.
